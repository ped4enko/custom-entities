== Description ==
A plugin for creating and managing entities (apps or bots) with rating capabilities.

== Installation ==
1. Upload the `custom-entities` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[entities]` shortcode to display the list of entities on pages or posts.

== Frequently Asked Questions ==

= How do I add entities? =
Go to the "Entities" section in the admin panel and add new entities.

= How do I display entities on a page? =
Use the `[entities]` shortcode in the content of a page or post.
